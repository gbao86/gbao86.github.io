# Trịnh Gia Bảo (gbao86) - Personal Portfolio 🚀

Trang web cá nhân trưng bày các dự án nổi bật, kỹ năng và kinh nghiệm của tôi trong lĩnh vực Full-Stack Web, Mobile (Android/Flutter) và tích hợp AI.

🔗 **Live Demo:** [https://gbao86.github.io](https://gbao86.github.io)

## 🛠 Tech Stack
Dự án được xây dựng hoàn toàn thuần tủy (Vanilla), không phụ thuộc vào framework nặng để tối ưu hóa hiệu suất và tốc độ tải trang:
* **HTML5 & CSS3:** Flexbox, CSS Grid, Custom Variables, Animations.
* **Vanilla JavaScript:** Canvas particles background, Scroll reveal effects, DOM manipulation.

## ✨ Điểm nổi bật
* **Giao diện Dark Theme hiện đại:** Kết hợp hiệu ứng hạt (particles) tương tác trôi nổi.
* **Responsive 100%:** Hiển thị hoàn hảo trên Mobile, Tablet và Desktop.
* **Tối ưu UX/UI:** Hiệu ứng cuộn mượt mà (smooth scrolling) và lazy-reveal.
* **Phân loại rõ ràng:** Portfolio chia mảng cụ thể (App Mobile, Web/Extension, Train AI).
